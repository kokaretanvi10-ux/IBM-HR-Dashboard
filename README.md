📊 IBM Employee Analytics Dashboard (Excel)

📌 Project Overview

This project presents an interactive Excel dashboard built using an IBM employee dataset. The dashboard focuses on analyzing employee-related metrics such as monthly income, performance ratings, and job roles to uncover meaningful insights.


The goal of this project is to transform raw HR data into a structured and visually engaging dashboard that supports data-driven decision-making in workforce management.


🎯 Objective


The primary objective of this dashboard is to analyze employee performance and compensation patterns within the organization. It helps in understanding how income varies across different roles and how performance ratings are distributed among employees.

This project also aims to demonstrate the use of Excel as a powerful tool for HR analytics and business insights generation.


🛠️ Tools and Techniques Used


The dashboard has been developed using Microsoft Excel with a focus on analytical and visualization techniques. Raw data was first cleaned and structured to ensure consistency. Pivot tables were used to summarize large datasets, and pivot charts were created to represent key metrics visually.


Interactive slicers were implemented to allow users to filter data based on job roles and performance ratings. Conditional formatting techniques were used to highlight key values and trends, improving readability and user experience.


📂 Dataset Description


The dataset consists of employee-level data with the following key fields:


Employee Number – Unique identifier for each employee

Monthly Income – Salary data used for compensation analysis

Performance Rating – Employee performance evaluation metric

Job Role – Employee designation (e.g., Research Scientist, Laboratory Technician, etc.)


This structured dataset enables detailed analysis of workforce distribution and performance.


📈 Dashboard Functionality


The dashboard provides a comprehensive and interactive view of employee data. Users can explore how salaries vary across different job roles and analyze the distribution of performance ratings.


With the help of filters and slicers, users can dynamically adjust the dashboard to focus on specific roles or performance categories. The visual elements such as bar charts and summary metrics make it easy to interpret complex data quickly.


🔍 Key Insights


The analysis highlights clear patterns in employee compensation and performance. It shows how certain job roles have higher average income levels, while others may have a wider distribution of salaries.


Performance ratings are also visualized to understand how employees are evaluated across the organization. This can help identify high-performing groups as well as areas that may require improvement.


The dashboard enables better understanding of the relationship between job role, salary structure, and employee performance.


💡 Skills Demonstrated


This project demonstrates strong capabilities in data analysis and dashboard creation using Excel. It reflects the ability to clean and organize raw data, perform meaningful analysis, and present insights in a visually appealing format.


It also showcases practical knowledge of HR analytics, data visualization, and business intelligence concepts.


📌 Conclusion

This project showcases how employee data can be effectively analyzed using Excel to generate actionable insights. The dashboard provides a clear understanding of workforce trends and helps in making informed decisions related to employee performance and compensation.

It is a strong example of combining data analytics with business understanding, making it highly relevant for roles in finance, analytics, and HR domains.
